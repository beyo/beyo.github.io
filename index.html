Features

Asynchronous

Using co as it's foundation, make use of generators, thunks, promises, and other asynchronous methods for maximum efficiency.

Modular

Build extremely modular applications; from using application modules, embedded or external plugins, to standard npm modules.

Structured

From small to large scale applications, take advantage of a HMVC architecture added an environment configuration aware without the bootstrap hassle.

Customizable

No obstruction on dependencies or unwanted features; start from a well defined application structure and create something great!
