Beyo

Small Application Framework, Great possibilities.

Documentation

This project is an active ongoing project. There will be some API changes over time. More documentation, and examples are to be written.

Therefore, and despite it working just fine, this is not a production-ready framework. All contributions welcome!

Features

Asynchronous

Using co as it's foundation, make use of generators, thunks, promises, and other asynchronous methods for maximum efficiency.

Modular

Build extremely modular applications; using application modules, embedded or external plugins, to standard npm modules.

Structured

From small to large scale applications, take advantage of a HMVC architecture with multi-level routing without the bootstrap hassle.

Customizable

No obstruction on dependencies or unwanted features; start from a basic MVC REST-enable HTTP server using koa and create something great!
